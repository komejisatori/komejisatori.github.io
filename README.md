# Simple Academic Homepage

This repository now contains a minimal static academic homepage that can be deployed directly with GitHub Pages.

## Files

- `index.html`: homepage structure
- `styles.css`: visual style and responsive layout
- `main.js`: small script for the footer year

## Quick customization

Replace the placeholder content in `index.html`:

- `Your Name`
- `Your Institution`
- `your.email@example.com`
- research topics
- publication entries
- external links such as GitHub and Google Scholar

## Deploy with GitHub Pages

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to `Pages`.
4. Set the source to deploy from the `main` branch (root).
5. Save and wait for GitHub Pages to publish the site.
